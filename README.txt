-----------------------------------README--------------------------------------
To run this Go program, Go must be installed on your system and the Root
and Path must be appropriately configured.
-------------------------------------------------------------------------------
1.  Open your Console or Console program (e.g. iTerm on MacOSX)
2.  Navigate to the appropriate directory where this Go program is located
    from the Console (e.g. cd Documents/Go Docs)
3.  Input the following statement into the Console command line:
		go run timeserver.go
3a. If desired, the following flags can be appended to the previous command
		--port (portvalue)
		-V
4.  Open your preferred web browser, and navigate to:
		http://localhost:(portvalue)/time
		NOTE: The default portvalue is 8080.

To terminate the program, use the following keystroke:  (control + c)
